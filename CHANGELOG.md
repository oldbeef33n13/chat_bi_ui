# Changelog

## 2026-03-06

### UX 优化（编辑器与运行态）
- 布局重构：编辑区从拥挤三栏升级为更聚焦的工作流布局，工具栏分区明确（公共操作/元素插入/预览与导出）。
- 工具栏交互：实现 `悬停预览 + 点击锁定` 双模式，统一菜单行为，降低误触与跳转成本。
- 直达插入：新增 `图表/表格/文本` 直达入口与画廊式选择，弱化“先选类型再添加”的繁琐路径。
- 上下文操作：节点头部改为悬浮操作条（编辑态/运行态一致），减少视觉噪音与常驻控件占位。
- 全屏沉浸：支持全屏预览、浮动半透明吸顶工具栏、鼠标唤起与自动隐藏，提升展示场景可读性。
- 报告体验：全屏报告按连续流展示（非强制分页翻页感），更接近阅读与汇报习惯。

### Inspector（属性面板）可用性
- 去掉“快捷/标准/专家”分层，改为统一结构化属性编辑，降低学习门槛。
- 支持文档全局属性编辑（Dashboard/Report/PPT），并补齐导出链路映射。
- 图表数据配置增强：支持 `多系列维度`（series/color/facet）可视化编辑与增删。
- 图表多 X 轴能力升级为真实并行：渲染输出 `xAxis[]` + `series.xAxisIndex`。
- 多 X 轴交互采用渐进式披露（Progressive Disclosure）：
- 默认单 X 轴，保持小白路径简洁。
- 高级区通过 `显示高级X轴` 展开。
- 提供 `一键自动匹配 X 轴` 快捷动作。
- 提供“当前映射”摘要：`指标 -> xAxis[n](field)`。

### 表格体验（多级表头/合并）
- 新增“多级表头设计器”：卡片化编辑 `rowSpan/colSpan/align`，操作更直观。
- 新增 `从列生成` 快捷动作，快速生成初始表头骨架。
- 新增 `表头实时预览`，即时校验合并结果。
- 保留“精确合并规则（可选）”用于复杂场景精细控制。

### 关键术语/关键操作
- 术语：`Progressive Disclosure`、`Hover Preview`、`Click Lock`、`Contextual Toolbar`、`Immersive Mode`、`xAxis[]`、`series.xAxisIndex`。
- 关键操作：`显示高级X轴`、`一键自动匹配 X 轴`、`+系列`、`从列生成`、`表头实时预览`、`图表/表格/文本直达插入`。
